# RefundStatus

This can be one of ["SUCCESS", "PENDING", "CANCELLED", "ONHOLD", "FAILED"]


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `RefundStatusSuccess`   | SUCCESS                 |
| `RefundStatusPending`   | PENDING                 |
| `RefundStatusCancelled` | CANCELLED               |
| `RefundStatusOnhold`    | ONHOLD                  |
