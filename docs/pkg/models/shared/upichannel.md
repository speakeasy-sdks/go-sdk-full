# UpiChannel

Specify the channel through which the payment must be processed. Can be one of ["link", "collect", "qrcode"]


## Values

| Name                | Value               |
| ------------------- | ------------------- |
| `UpiChannelLink`    | link                |
| `UpiChannelCollect` | collect             |
| `UpiChannelQrcode`  | qrcode              |
